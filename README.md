
# COFFEE_RANGE

The Coffee Range Application is an e-commerce platform developed for a coffee shop named Coffee Range. This application utilizes the MERN stack for development:

Frontend: React 

Backend: Express.js

Database: MongoDB

Server: Node.js

The application provides users with an intuitive interface to browse and purchase various coffee products.
